# CS471 – Lab 2
## Django and Basic Git Operations

This repository contains the implementation of **Lab 2** for the course  
**CS471 – Web Technologies**.

### Lab Objectives
- Install and configure the Django web framework
- Create and organize a Django project structure
- Create Django applications (modules)
- Use basic Git commands
- Connect a local repository to GitHub

### Project Structure
- `libraryproject/` – Main Django project
- `apps/bookmodule/` – Book service module
- `apps/usermodule/` – User service module
- `.gitignore` – Git ignore rules

### How to Run
```bash
python manage.py runserver
```
Then open:
```bash
http://127.0.0.1:8000/
```
